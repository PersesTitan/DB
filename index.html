데이터베이스 수행

데이터베이스 정의

  • 데이터들의 집합

파일 시스템 단점(5가지)

  • 테이터 불일치 현상
  • 데이터를 중복 저장하므로 기억 장소 낭비
  • 공유해서 사용할 수 없다
  • 보안 조치가 미흡하다
  • 원하는 데이터 작업을 쉽게 할 수 있는 명령어(질의어)가 없다.

데이터베이스 특징(4가지)

  • 실시간 접근성
  • 실시간 변화(지속적인 변화)
  • 내용에 의한 참조
  • 동시 공유

DBMS(3가지)

  • Oracle
  • MySQL
  • Maria DB

데이터베이스 종류(3가지)

  • 계층형 데이터베이스
  • 네트워크형(망형) 데이터베이스
  • 관계형 데이터베이스

정보구축 5단계

  • 분석
  • 설계
  • 구현
  • 테스트
  • 유지, 보수

E-R 다이어그램 표시 기호(4가지)

  • 개체

  • 속성

  • 관계

  • 서로 간의 관계

데이터베이스 필수 용어(4가지)

  • DBMS
  • 기본키
  • 외래키
  • SQL

SQL 그룹 3가지

  • DDL
  • DML
  • DCL

DDL 명령어

  • create
  • alter
  • drop

DML 명령어

  • insert
  • select
  • update
  • delete

DCL 명령어

  • commit
  • rollback
  • grant
  • revoke
